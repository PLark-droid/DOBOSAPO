# DOBOSAPO セミナー資料

ドボサポセミナーの資料管理リポジトリです。

## プロジェクト構成

```
DOBOSAPO/
├── slides/          # スライド資料
├── handouts/        # 配布資料
├── code-samples/    # サンプルコード
├── images/          # 画像素材
├── resources/       # 参考資料
└── docs/            # ドキュメント
```

## ディレクトリ説明

### slides/
プレゼンテーション用のスライド資料を格納します。
- Markdown形式
- PowerPoint形式
- reveal.js等のHTML形式

### handouts/
参加者向けの配布資料を格納します。
- PDF資料
- ワークシート
- チェックリスト

### code-samples/
セミナーで使用するサンプルコードを格納します。
- デモコード
- 演習用コード
- ソリューション

### images/
スライドや資料で使用する画像ファイルを格納します。
- 図表
- スクリーンショット
- ロゴ等

### resources/
参考資料やリンク集を格納します。
- 参考文献リスト
- 関連リンク
- 補足資料

### docs/
セミナーに関する各種ドキュメントを格納します。
- 開催要項
- タイムテーブル
- Q&A集

## セットアップ

```bash
# リポジトリのクローン
git clone <repository-url>

# ディレクトリへ移動
cd DOBOSAPO
```

## 資料の作成

各ディレクトリに適切な形式でファイルを配置してください。

## バージョン管理

- 資料の更新は適切にコミットしてください
- ブランチを活用して複数バージョンを管理できます
