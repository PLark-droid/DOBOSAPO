# DOBOSAPO セミナー資料

## 建設業DX実践セミナー - Larkで実現する業務改革

地方中小建設業の経営者・管理者向けに、オールインワンアプリ「Lark」を活用した業務改革の実践方法をお伝えするセミナーの資料管理リポジトリです。

### セミナーコンセプト
元建設業跡取りが、自社でのLark導入経験を通じて得た知見を共有。建設業特有の課題に対して、Larkを中心としたDX推進の実践的アプローチを解説します。

## プロジェクト構成

```
DOBOSAPO/
├── slides/          # スライド資料
├── handouts/        # 配布資料
├── code-samples/    # サンプルコード
├── images/          # 画像素材
├── resources/       # 参考資料
└── docs/            # ドキュメント
```

## ディレクトリ説明

### slides/
プレゼンテーション用のスライド資料を格納します。
- Markdown形式
- PowerPoint形式
- reveal.js等のHTML形式

### handouts/
参加者向けの配布資料を格納します。
- PDF資料
- ワークシート
- チェックリスト

### code-samples/
セミナーで使用するサンプルコードを格納します。
- デモコード
- 演習用コード
- ソリューション

### images/
スライドや資料で使用する画像ファイルを格納します。
- 図表
- スクリーンショット
- ロゴ等

### resources/
参考資料やリンク集を格納します。
- 参考文献リスト
- 関連リンク
- 補足資料

### docs/
セミナーに関する各種ドキュメントを格納します。
- 開催要項
- タイムテーブル
- Q&A集
