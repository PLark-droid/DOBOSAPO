# セミナー開催計画

## 基本情報
- セミナー名: **建設業DX実践セミナー - Larkで実現する業務改革**
- 対象者: 地方中小建設業の経営者・管理者・DX推進担当者
- 定員: 20-30名
- 所要時間: 15分（コンパクト版）
- 形式: オンライン or オフライン（ハイブリッド可）
